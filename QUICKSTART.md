# ⚡ БЫСТРЫЙ СТАРТ (30 МИНУТ)

## 📦 ЧТО ТЫ ПОЛУЧИШЬ

Telegram бот, который:
- ✅ Ведёт финансовый учёт (доходы/расходы)
- ✅ Управляет задачами для партнёров
- ✅ Анализирует цены и наценки
- ✅ Общается как личный ассистент
- ✅ Работает 24/7 в облаке

---

## 🚀 ПЛАН ДЕЙСТВИЙ

### ВАРИАНТ 1: САМЫЙ БЫСТРЫЙ (Railway - облако) - 10 минут

#### Шаг 1: Скачай 4 файла (1 мин)
Скачай из папки с этой инструкцией:
- `gze_bot.py`
- `requirements.txt`
- `Procfile`
- `.env`

#### Шаг 2: GitHub (2 мин)
1. Открой https://github.com/new
2. Создай новый репозиторий "GZE-Bot"
3. Нажми "Create repository"

#### Шаг 3: Загрузи файлы на GitHub (3 мин)
Вариант А (проще):
1. После создания нажми "uploading an existing file"
2. Перетащи 4 файла
3. Нажми "Commit changes"

Вариант Б (правильнее):
1. Установи GitHub Desktop https://desktop.github.com/
2. Создай локальный репозиторий
3. Положи 4 файла в папку
4. Нажми "Commit" → "Push"

#### Шаг 4: Railway (4 мин)
1. Открой https://railway.app/
2. "Sign Up" → "Continue with GitHub" → разреши доступ
3. "+ New Project" → "Deploy from GitHub"
4. Выбери репозиторий "GZE-Bot"
5. Railway автоматически запустит бот 🎉

---

### ВАРИАНТ 2: КЛАССИЧЕСКИЙ (локальный) - 5 минут

Если хочешь сначала протестировать на своём компьютере:

1. **Установи Python 3.9+** https://www.python.org/

2. **Открой командную строку** и выполни:
```bash
pip install -r requirements.txt
```

3. **Запусти тест:**
```bash
python test_bot.py
```

Если всё зелёное ✅ - переходи к Варианту 1 (Railway)

## 🔍 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

**Бот не отвечает:**
- Проверь интернет
- Перезагрузи Railway (Dashboard → Redeply)
- Посмотри логи (Railway → Logs)

**Ошибка с ключами:**
- Убедись что .env правильный
- Скопируй ещё раз из письма выше

**Ошибка "не найден Python":**
- Переустанови Python
- Выбери "Add Python to PATH"
